## Mac的快捷键基本用法

### 1.关于Vscode的终端

- Ctrl + ` 显示终端
- Ctrl + Shift + ` 新建终端
- Ctrl + Cmd + up 终端向上移动
- Command + Up 向上滚动
- Command + Down 向下滚动
- PgUp 向上翻页
- PgDown 向下翻页
- Command + Home 滚动到顶部
- Command + End 滚动到底部

### 2.关于浏览器

- Ctrl + Shift + Tab 切换标签页


### 3.关于应用程序

- Cmd + Shift + Tab 切换应用程序
- Ctrl + right/left 切换应用程序
